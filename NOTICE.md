# NOTICE — heirloom-citations-darwin

## Scope

Curated reference-documentation collection.  Primary sources for
the Heirloom Darwin port cited by `BIBLIOGRAPHY.md` in each of the
sibling code repos.

**Not authoritative.** Every PDF, PostScript file, or other artefact
in this repo is preserved upstream at the TUHS Unix Heritage
Archive (<https://www.tuhs.org/Archive/>). This repo is a curated
subset for offline / air-gapped / self-contained deployment.

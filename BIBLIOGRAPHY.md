## How to cite this repository

Do NOT cite `heirloom-citations-darwin` as a primary source.  Cite
the underlying document, and if the specific PDF you consulted
matters, cite via TUHS Archive URL:

```
Ossanna, Joseph F. (1977). "NROFF/TROFF User's Manual."
Bell Laboratories Computing Science Technical Report No. 54.
Retrieved via https://www.tuhs.org/Archive/Documentation/TechReports/Bell_Labs/CSTRs/54.pdf
```

The heirloom-citations-darwin repo is a mirror, not a publisher.
